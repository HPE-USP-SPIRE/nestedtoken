# nestedtoken

This package implements the nested token model, as defined in https://docs.google.com/document/d/1nQYV4wf8wiogpxboIVbwtFZyZjLNRejyguHoGZIZLQM

To use:  
import "github.com/hpe-usp-spire/nestedtoken" in go file  
go get github.com/hpe-usp-spire/nestedtoken to add in go.mod  

Functions description are in go file.
